Dodge the Creeps is a simple, fast-paced 2D survival game built with the Godot Engine. The objective is to avoid the enemies for as long as possible and get the high score!

## Gameplay

The rules are simple:
* You control the player character.
* Enemies ("Creeps") will spawn from the edges of the screen and move in straight lines.
* If a Creep touches you, the game is over.
* Your score increases for every second you manage to survive.
